フォントをダウンロードして、app/javascript/src配下に置く

app/javascript/src/style.scssにフォントのパスを記入と使用

```scss
@font-face {
  font-family: "maru-gosic";
  src: url('./Corporate-Logo-Rounded.ttf') format('truetype');
}

html {
  font-family: "maru-gosic";
}
```

これで、OK!!
